Overview

What is this project?

A production-grade cloud infrastructure running locally, demonstrating real-world DevSecOps practices.

🔐

Role-Based Access

Three-tier RBAC system with Viewer, Editor and Admin roles enforced via Spring Security @PreAuthorize annotations on every endpoint.

☁️

Cloud Emulation

LocalStack emulates AWS services locally — S3 bucket for file storage and DynamoDB tables for user and role management. No real AWS account needed.

🏗️

Infrastructure as Code

All cloud resources are defined in Terraform. Running terraform apply provisions S3, DynamoDB tables and seed data automatically.

🔍

Full Observability

Prometheus scrapes metrics every 15s from Spring Actuator. Grafana displays a live dashboard of authorization errors (HTTP 403) and all request rates.
